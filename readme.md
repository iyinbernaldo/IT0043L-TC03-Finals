## Overview
A collection of pages creating a whole simplistic and minimalistic website. This was created to feature a portfolio of past works and a resume.
The whole website is created using HTML, CSS, and JavaScript.

### Key Components
- MultiPage Website
- Parallax transition
- CSS Animation
- JavaScript script
